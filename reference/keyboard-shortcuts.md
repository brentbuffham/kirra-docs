# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in Kirra.

---

## File Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New blast |
| `Ctrl+O` | Open blast file |
| `Ctrl+S` | Save |
| `Ctrl+Shift+S` | Save As |
| `Ctrl+I` | Import |
| `Ctrl+E` | Export |
| `Ctrl+P` | Print / PDF Export |
| `Ctrl+W` | Close current project |

---

## Edit Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Ctrl+Shift+Z` | Redo (alternate) |
| `Ctrl+A` | Select All |
| `Ctrl+D` | Deselect All |
| `Delete` | Delete selected holes |
| `Ctrl+C` | Copy selected holes |
| `Ctrl+V` | Paste |
| `Ctrl+X` | Cut selected holes |
| `Ctrl+F` | Find / Select holes by property |

---

## Tools

| Shortcut | Action |
|----------|--------|
| `S` | Switch to Select tool |
| `A` | Switch to Add Hole tool |
| `G` | Open Pattern Generator |
| `M` | Move selected holes by offset |
| `T` | Toggle Timing panel |
| `C` | Toggle Charging panel |
| `Escape` | Cancel current action / deselect |

---

## View / Navigation

| Shortcut | Action |
|----------|--------|
| `+` or `=` | Zoom in |
| `-` | Zoom out |
| `F` | Fit all holes in viewport |
| `Space + drag` | Pan (hold Space, drag mouse) |
| `Ctrl+0` | Reset zoom to 100% |
| `Ctrl+Shift+F` | Toggle fullscreen |

---

## Analysis Panels

| Shortcut | Action |
|----------|--------|
| `Shift+S` | Toggle Blast Statistics panel |
| `Shift+V` | Toggle Voronoi panel |
| `Shift+P` | Toggle PPV Analytics panel |

---

## Hole Selection

| Shortcut | Action |
|----------|--------|
| `Click` | Select single hole |
| `Ctrl+Click` | Add/remove hole from selection |
| `Drag` | Box-select (rubber-band selection) |
| `Ctrl+A` | Select all holes |
| `Ctrl+D` | Deselect all |

---

## Canvas Helpers

| Shortcut | Action |
|----------|--------|
| `Ctrl+G` | Toggle snap-to-grid |
| `Ctrl+L` | Toggle hole ID labels |
| `Ctrl+T` | Toggle timing number display |
| `Ctrl+K` | Toggle charge colour mode |

---

## Timing Panel

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+T` | Apply timing to selection |
| `Ctrl+Shift+C` | Clear timing from selection |

---

## Miscellaneous

| Shortcut | Action |
|----------|--------|
| `F1` | Open help documentation |
| `F5` | Refresh / recalculate all |
| `F11` | Toggle fullscreen |

---

> **Tip:** Most toolbar buttons show their keyboard shortcut in the tooltip when you hover over them.
