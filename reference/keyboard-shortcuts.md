# Keyboard Shortcuts & Mouse Controls

Quick reference for keyboard shortcuts and mouse controls in Kirra Design.

---

## Toggle Keys (ON/OFF)

| Key | Action |
|-----|--------|
| **F** | Hole bearing focus mode (when Bearing tool is active) |
| **S** | Self-snap toggle |
| **Shift** | Multi-select mode |

---

## Actions

| Shortcut | Action |
|----------|--------|
| **Ctrl+Z** | Undo |
| **Ctrl+Y** or **Ctrl+Shift+Z** | Redo |
| **Escape** | Clear selections and reset tools |
| **Backspace** / **Delete** | Delete selected KAD objects/vertices |
| **Backspace** / **Delete** | Delete last drawn point (when drawing) |

---

## 3D Mouse Modifiers

| Action | Result |
|--------|--------|
| **Alt + drag** | Orbit camera |
| **Alt + Shift + drag** | Camera roll |
| **Scroll wheel** | Zoom |
| **Right-click** | Context menu (no camera movement) |

---

## General Navigation

| Action | Result |
|--------|--------|
| **Scroll wheel** | Zoom in/out (2D and 3D) |
| **Click + drag** | Pan (default mode) |
| **Click** | Select single entity |
| **Shift + click** | Add to selection |
