# Frequently Asked Questions

---

## General

### What browser should I use?

Chrome is recommended. Kirra uses WebGL and Web Workers for 3D rendering and analysis.

### Is my data stored on a server?

No. All data is stored locally in your browser's IndexedDB. Nothing is sent to any server.

### Can I use Kirra offline?

Yes, once loaded. Kirra runs entirely in the browser.

### How do I switch between 2D and 3D?

Use the 2D/3D toggle or the view buttons in the toolbar.

### My coordinates look wrong — what coordinate system does Kirra use?

Kirra uses UTM-style real-world coordinates (X=East, Y=North, Z=Elevation). See [Coordinate System](coordinate-system.md).

### How do I import from Surpac?

File > Import, select your .dtm and .str file pair. Note Surpac uses Y,X (Northing, Easting) ordering.

### What file formats does Kirra support?

20+ formats including CSV, DXF, Surpac DTM/STR, OBJ, PLY, GLTF/GLB, IREDES XML, KML, Shapefile, LAS, and more.

### Is Kirra free?

Free for mining, quarrying, construction professionals and researchers. Commercial software integration requires a licence. See the Kirra Licence.

### How do I report a bug?

Click the "Report" button in the Welcome popup, or email blastingapps.xyz@gmail.com.
