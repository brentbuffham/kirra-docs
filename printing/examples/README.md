# Template Examples

Downloadable XLSX example templates for Kirra's Print from Template system.

| File | Description |
|------|-------------|
| `01-Kirra_Print_Sheets.xlsx` | General-purpose print sheet layout |
| `Voronoi-Showcase.xlsx` | All 10 Voronoi metrics across 3 legend modes (default, fixed, min-max) |
| `HoleSection-Grid.xlsx` | `[+>]` grid pagination with holeSection, filter, and math-wrapped fields |
| `MultiMap-Views.xlsx` | Four-quadrant layout with different display code combinations |
| `Charging-Report.xlsx` | Charging summaries, product breakdowns, grouped statistics |
| `HoleData-Table.xlsx` | Per-hole tables with `[++]`, aggregations, and filtered subsets |

See [Template Examples](../template-examples.md) for detailed walkthroughs of each template.

See [Print Formula Reference](../pdf-print.md) for the complete list of all variables, functions, and display codes.
