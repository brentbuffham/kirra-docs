# Theming

Kirra Scheduler supports three visual modes, all toggled from the header. Your preference is automatically saved and restored between sessions.

> *Screenshot coming soon*

---

## Dark Mode (Default)

Dark backgrounds with light text. Designed for extended use in mine site offices and control rooms where reduced screen glare is preferred.

- Background: dark greys
- Text: light grey / white
- Accent colours: teal (prep), blue (drill), amber (load), red (blast)

---

## Light Mode

Toggled via the **sun/moon icon** in the header. Uses softened whites (slightly darker than pure white) to reduce glare.

- Background: soft greys
- Text: dark grey / black
- Same accent colour palette as dark mode

---

## Colourblind Mode

Toggled via the **CB** button (eye icon) in the header. Applies a deuteranopia-safe palette that replaces red/green distinctions with more easily differentiated colours:

| Normal Colour | Colourblind Replacement | Usage |
|---|---|---|
| Red | Magenta / pink | Blast accent, breach warnings |
| Green | Blue | OK status, dependency connectors |

Colourblind mode works in combination with either dark or light mode.

---

## Accent Colours

The following accent colours are used throughout the application for consistent visual identification:

| Colour | Usage |
|---|---|
| **Teal** | Pattern Preparation phase |
| **Blue** | Drilling phase |
| **Amber/Gold** | Loading phase |
| **Red** | Blasting phase / warnings |
| **Green** | OK status / met dependencies |

Dependency connectors automatically adapt their colours to match the current theme and colourblind settings.

---

## Brand Colours

The application title gradient and active tab highlight use a red accent colour, derived from the Kirra application's branding.
